## Quick Reference
- `shortcode:accordion` embeds a collapsable panel. [» Full instructions](#accordion)
- `shortcode:accordionitem` item embeds a additional item to the accordion. [» Full instructions](#accordion)
- `shortcode:chart` embeds a ChartJS chart. [» Full instructions](#chart)
- `shortcode:map` embeds Google Maps. [» Full instructions](#map)
- `shortcode:mapstatic` embeds a static Google Map. [» Full instructions](#map)
- `shortcode:panel` embeds a bootstrap panel. [» Full instructions](#panel)
- `shortcode:parallax` embeds a parallax image section. [» Full instructions](#parallax)
- `shortcode:vimeo` embeds a Vimeo video. [» Full instructions](#vimeo)
- `shortcode:youtube` embeds a YouTube video. [» Full instructions](#youtube)


## Accordion
`shortcode:accordion`

```
{[ accordion ]}
  [[ item title:'Small Groups' ]]
    
  [[ enditem ]]
  
{[ endaccordion ]}
```

`shortcode:accordionitem`

```
[[ item title:'Small Groups' ]]
  
[[ enditem ]]
```

## Chart
`shortcode:chart`
```
{[ chart type:'bar' ]}
[[ dataitem label:'Small Groups' value:'45' ]] [[ enddataitem ]]
  
{[ endgooglemap ]}
```

## Map
`shortcode:map`

```
{[ googlemap height:'600px' width:'100%' showzoom:'true' showstreetview:'true' showfullscreen:'true' showmapttype:'true' scrollwheel:'true' draggable:'true' ]}
[[ marker location:'10451 W Palmeras Dr Sun City, AZ 85373-2000' ]] [[ endmarker ]]
  
{[ endgooglemap ]}
```

`shortcode:mapstatic`

```
{[ googlestaticmap center:'10451 W Palmeras Dr Sun City, AZ 85373-2000' zoom:'12' ]}
{[ endgooglestaticmap ]}
```

## Panel

`shortcode:panel`

```
{[ panel  title:'panel title' icon:'fa fa-star' ]}
  
{[ endpanel ]}
```

## Parallax

`shortcode:parallax`

```

{[ endparallax ]}
```
