## Control Flow Tag

<Row>
  <Column>

|    Prefix | Method                                                                                  |
| --------: | --------------------------------------------------------------------------------------- |
|      `if` | <code>{% if condition %} ... {% endif %}</code>                                                    |
|    `else` | <code>{% else %}</code>                                                                            |
|  `elseif` | <code>{% elseif condition %}</code>                                                                |
|  `ifelse` | <code>{% if condition %} ... {% else %} ... {% endif %}</code>                                     |
|  `unless` | <code>{% unless condition %} ... {% endunless %}</code>                                            |
|    `case` | <code>{% case variable %} ... {% when condition %} ... {% else %} ... {% endcase %}</code>         |
|    `when` | <code>{% when condition %}</code>                                                                  |
|     `if-` | <code>{%- if condition -%} ... {%- endif -%}</code>                                                |
|   `else-` | <code>{%- else -%}</code>                                                                          |
| `elseif-` | <code>{%- elseif condition -%}</code>                                                              |
| `ifelse-` | <code>{%- if condition -%} ... {%- else -%} ... {%- endif -%}</code>                               |
| `unless-` | <code>{%- unless condition -%} ... {%- endunless -%}</code>                                        |
|   `case-` | <code>{%- case variable -%} ... {%- when condition -%} ... {%- else -%} ... {%- endcase -%}</code> |
|   `when-` | <code>{%- when condition -%}</code>                                                                |

  </Column>
</Row>

## Iteration Tag

|         `for` | <code>{% for item in collection %} … {% endfor %}</code>                                |

|   `continue-` | <code>{%- continue -%}</code>                                |

  </Column>
</Row>
