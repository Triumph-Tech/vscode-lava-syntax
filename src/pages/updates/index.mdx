## Upcoming Features

- **Intellisense Features**
  - **Hover Support**. Hovering over any filter will provide instant contextual information.
  - **Auto Completion**. Auto complete all properties from Rock, no more guessing column and table names!
- **Lava Document Formatter**.
- **Linting**.
- **Error Detection**.

### Ongoing Fixes
- Adding new syntax as features are added to Rock.
- Documentation Updates.

## Changelog

### Version 1.0.5 (December 14, 2020)

**New Features**

- Added snippets for AddResponseHeader, WriteCookie, ReadCookie, ImageUrl and RockInstanceConfig.

**Improvements**

- Added validation on Lava Shortcode parameters. (Disallowing uppercase characters)
- Fixed Error on Google Shortcode map.


### Version 1.0.4 (December 2, 2020)

**New Features**

- Next generation Lava comment highlighting
- Calendar Events Command
- Event Scheduled Instance Command
- Interaction Content Channel Item Write Command
- Interaction Write Command

**Improvements**

- Improved brackets and quotes inside comments.
- Fixed Address filter from incorrectly rendering options.
- Fixed filters where true/false was rendered without showing choice dropdown.

### Version 1.0.3 (October 28, 2020)
- Fixed date snippet tabbing.
- Improved entity command markup to include security enabled defaults.
- Fixed Lava file defaults to conform to the [Lava Style Guide](https://community.rockrms.com/lava/style)
- Added increment and decrement snippets.
- Added pagination snippet.
- Improved public landing pages and launched documentation site.

### Version 1.0.2 (October 28, 2020)
- Shortlived release replaced by 1.0.3

### Version 1.0.1 (July 8, 2020)
- Fixed code expansion. Allowing snippets to be used inside of other snippets.
- Added basic XAML Language Support
- Added `get_` shortcodes to retrieve images, communications, and files.


### Version 1.0 (April 1, 2020)
- Complete rewrite of code highlighting, which added support for inline C++, SQL, CSS, and LESS code.
- Merged Lava Snippets extension into this extension.
